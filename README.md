##### 우리가 자주 가는 우리의 포털 들에게,,Our Portal
> 우리가 자주 쓰는 사이트 중에는 다른 기능을 하는 사이트들도 있고, 비슷한 기능을 하지만 상황에 따라 각자 다른 장점을 갖고 있는 사이트도 있다. 나도 이런 점을 잘 알고, 그 순간 원하는 사이트에 들어가서 적절한 검색을 하고 싶지만 매번 아무 생각 없이 크롬에 켜진 메인 구글 사이트에서 검색을 시작하는 실수를 한다:sweat_drops:.  
> 그래서 그냥 애초에 한 군데에서 검색을 하게 하고, 각 사이트 쿼리 규칙을 찾아서, 검색어가 입력된 사이트를 쭉 모아 놓으면 맨번 하는 실수가 없어지지 않을까:laughing:!?라는 생각이 들었다. React기초 강의 들은지도 얼마 안 됐고, 간단한 사이트니 배운걸 이용해 보면 좋을거 같아서 리액트를 사용해 만들어 보았다.
